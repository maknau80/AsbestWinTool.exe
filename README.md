# Утилита *AsbestWinTool.exe* от Asbest Software предназначена для кастомизации («косметического» изменения) системной информации Windows и управления Microsoft Store
## Кастомизация системной информации (WinVer) позволяет изменить отображаемые данные о версии Windows:
```
>_ Изменение названия ОС

>_ Редактирование OEM-информации

>_ Подмена winver.exe
```
> <img width="458" height="410" alt="image" src="https://github.com/user-attachments/assets/0ec6855b-22b1-44ef-a184-245f5ab6f99e" />

## Изменение данных о «железе»
```
>_ В разделе «Install PC Components» скрипт позволяет вручную отредактировать ключи реестра,
   отвечающие за названия комплектующих в диспетчере устройств:

 * Процессор (CPU)
 * Видеокарта (GPU)
```

## Управление Microsoft Store
```
>_ Установка: Перерегистрация пакета Microsoft Store, если он был удален или поврежден.

>_ Удаление: Полная деинсталляция магазина из системы для всех пользователей.
```
> <img width="518" height="188" alt="image" src="https://github.com/user-attachments/assets/312f569f-6735-4c8e-b8ab-376a10f1f558" />

## Резюме
### Это инструмент для создания «персонализированных» сборок Windows или для шуток над пользователями (изменение названия комплектующих в системе на несуществующие).
